## 📖 Descripción

**BaiMai Music** es una aplicación de streaming de música de código abierto desarrollada por **The BaiMai Open Network**. Diseñada con un enfoque en la **privacidad**, **seguridad** y **experiencia del usuario**, te permite disfrutar de tu música favorita sin anuncios, sin rastreadores y sin comprometer tus datos.

## ✨ Características

### 🎧 Reproducción y Streaming
- Streaming desde YouTube/YouTube Music
- Reproducción en segundo plano
- Control desde notificaciones y pantalla de bloqueo
- Cola de reproducción inteligente

### 📥 Descargas y Modo Offline
- Descarga de canciones para reproducción offline
- Gestión de almacenamiento inteligente
- Calidad de audio configurable

### 🎨 Personalización
- Tema morado por defecto (personalizable)
- Modo claro / oscuro / sistema
- Selector de colores personalizados
- Animaciones fluidas

### 🌍 Multilingüe (18 idiomas)
🇪🇸 Español · 🇺🇸 English · 🇵🇹 Português · 🇫🇷 Français · 🇨🇳 中文 · 🇷🇺 Русский · 🇯🇵 日本語 · 🇩🇪 Deutsch · 🇮🇹 Italiano · 🇮🇳 हिन्दी · 🇰🇷 한국어 · 🇹🇭 ไทย · 🇮🇩 Bahasa Indonesia · 🇸🇪 Svenska · 🇫🇮 Suomi · 🇳🇴 Norsk · 🇻🇳 Tiếng Việt · 🇵🇭 Filipino

### 🔒 Privacidad y Seguridad
- Sin anuncios
- Sin registro ni login
- Sin rastreadores
- Almacenamiento local
- Código abierto

## 🚀 Instalación

### Android
Descarga el APK desde [Releases](https://github.com/TheBaiMaiOpenNetwork/BaiMai-Music/releases)

### Windows
Descarga el ZIP desde Releases, descomprime y ejecuta `baimai_music.exe`

### Linux (Ubuntu/Debian/Mint/Zorin)
```bash
sudo dpkg -i baimai-music-*.deb
```

## 🛠️ Desarrollo

### Requisitos
- Flutter 3.27+
- Dart 3.3+

### Instalación
```bash
git clone https://github.com/TheBaiMaiOpenNetwork/BaiMai-Music.git
cd BaiMai-Music
flutter pub get
flutter run
```
