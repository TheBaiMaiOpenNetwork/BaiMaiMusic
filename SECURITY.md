# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BaiMai Music, please report it by opening an issue at:
https://github.com/TheBaiMaiOpenNetwork/BaiMai-Music/issues

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond as quickly as possible and aim to fix critical issues within 7 days.

## Privacy Commitment

BaiMai Music is designed with privacy in mind:
- No personal data collection
- No user accounts or registration required
- No third-party trackers or analytics
- All data stored locally on your device
- Open source and auditable
